== Description ==

Creates a custom registration form and a members gallery that displays members gravarats and their name.

== Installation ==

1. Unzip the archive and put the `subscriber-gallery` folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin through the 'Plugins' menu in WordPress.

= Usage =

Install the plugin and it automatically creates the registration and members gallery pages. You need to manually create an author.php page. Use the code included in this plugin to create this page.
